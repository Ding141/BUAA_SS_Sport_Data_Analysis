# 基于时域特征的人体活动识别（HAR）

## 一、项目概述

使用UCI HAR数据集，从智能手机加速度计原始信号中提取时域特征，构建决策树模型对人体6种活动进行分类识别。

**活动类型**：走路、上楼、下楼、坐着、站着、躺着

## 三、数据集说明

| 项目 | 说明 |
|------|------|
| 来源 | UCI HAR Dataset |
| 设备 | 三星Galaxy S II智能手机 |
| 位置 | 固定在腰部 |
| 采样率 | 50 Hz |
| 窗口长度 | 2.56秒（128个时间步） |
| 训练集 | 7352样本（21人） |
| 测试集 | 2947样本（9人） |

**原始信号文件（9个）**：
- total_acc_x/y/z：总加速度（含重力）
- body_acc_x/y/z：身体加速度（去重力）
- body_gyro_x/y/z：陀螺仪角速度

---

## 四、提取的时域特征（6个）

| 特征 | 计算公式 | 物理意义 |
|------|----------|----------|
| PeakX | max(x) - min(x) | X轴冲击强度，区分上楼/走路 |
| StdMean | (std_x+std_y+std_z)/3 | 整体波动程度，区分动态/静态 |
| RMS | √(rms_x²+rms_y²+rms_z²) | 合向量有效值，反映活动强度 |
| MeanX | mean(x) | 重力方向基线，区分站/坐/躺 |
| SMA | mean(|x|+|y|+|z|) | 总活动量 |
| ZCR_X | 过零次数 / 128 | X轴振荡频率，反映步频 |

---

## 五、算法流程

1. 数据加载：读取9个原始信号文件（每行128个时间步）
2. 特征提取：对每个样本的X/Y/Z三轴计算6个时域特征
3. 数据划分：训练集7352样本，测试集2947样本
4. 模型训练：决策树（CART算法，基尼系数，max_depth=5）
5. 模型评估：测试集预测，计算准确率和混淆矩阵

---

## 六、实验结果

| 指标 | 结果 |
|------|------|
| 测试集准确率 | 70-75% |
| 最佳区分特征 | PeakX / StdMean |
| 最易混淆活动 | 坐着 vs 站着 |

**特征重要性排序**：
1. PeakX（X轴峰峰值）
2. StdMean（三轴平均标准差）
3. RMS（合向量有效值）
4. MeanX（X轴均值）
5. SMA（总活动量）
6. ZCR_X（X轴过零率）

---

## 八、结论

1. 时域特征能有效区分动态活动（走路系列）和静态活动（坐/站/躺）
2. 峰峰值和标准差是区分能力最强的两个特征
3. 区分上楼和下楼需要组合多个特征
4. 坐着和站着信号特征相似，需要引入频域特征或陀螺仪数据进一步区分
