# 基础路线：时域特征提取与分类

**对应大作业报告**：第 2 节

## 概述

从 UCI HAR 原始加速度信号（每窗口 128 点 × 3 轴）提取具有明确物理含义的时域统计特征，训练决策树和 GMM 分类器对 6 种活动进行识别。

## 核心 6 维时域特征

| 特征 | 公式 | 物理含义 | 区分能力 |
|------|------|---------|---------|
| PeakX | max − min | 冲击幅度 | 细分行走类 |
| StdMean | mean(σx, σy, σz) | 动态/静态区分 | **最强**（差一个数量级） |
| RMS | √(rms²x + rms²y + rms²z) | 总能量强度 | 与 StdMean 相关 >0.9 |
| MeanX | mean(sx) | 重力方向 | 完美分离躺卧 (~0.75g) |
| SMA | mean(\|sx\|+\|sy\|+\|sz\|) | 总活动量 | 上楼最高，躺卧最低 |
| ZCR_X | 符号变化/N | 步频粗糙代理 | 动静差一个数量级 |

## 分类器

### 决策树 (CART)
- 分裂准则：基尼系数
- 配置：max_depth=5, min_samples_split=20, min_samples_leaf=10
- 准确率：**~74%**
- 树结构揭示物理直觉：根节点 MeanX（重力方向）→ 二级 PeakX（冲击幅度）→ 三级 RMS / SMA / MeanX

### 高斯混合模型 (GMM)
- 生成式建模 + EM 参数估计 + BIC 自动选参
- 准确率：**~70%**
- 局限：概率密度重叠区偏向训练样本更多的类别

## 时域特征的根本局限

1. 静态活动混淆：SITTING 与 STANDING 在所有时域统计量上几乎完全重叠
2. 频率信息缺失：时域统计量只回答"信号波动多大"，不回答"以什么频率波动"

→ 这正是进阶路线引入频域分析的动机。
