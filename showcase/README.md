# 运动传感器时域与频域特征分析

> **信号与系统课程展示专项** | UCI HAR Dataset | 50 Hz, 128 点/窗

---

## 概述

本项目使用手机加速度计与陀螺仪数据，对六种人体运动状态（行走、上楼、下楼、静坐、站立、躺卧）进行**时域特征提取**与**频域特征分析**，最终通过决策树完成自动分类。

**核心命题：** 频域分析（FFT / STFT）能否提供时域统计量无法捕获的判别信息？

---

## 快速运行

```bash
cd showcase
python pipeline.py
```

输出：
- `showcase/figures/` 下 7 张可视化图片
- 终端打印分类准确率、消融实验、特征重要性

---

## 流水线结构

```
数据加载 → 时域波形可视化
         → STFT 频带能量时间演进
         → FFT 幅度谱叠加对比
         → 频带能量分布
         → 特征提取 (8频域 + 4时域 /通道 × 6通道 = 72维)
         → 决策树分类 (GridSearchCV)
         → 频谱特征散点图 / 特征重要性 / 混淆矩阵
```

## 图片说明

### 图 1 — 六种动作时域波形

![时域波形](figures/01_时域波形对比.png)

六种动作在 2.56 s 窗口内的加速度计三轴（X 红 / Y 绿 / Z 蓝）时域信号。动态类（行走/上楼/下楼）呈周期性震荡（振幅 ±5g），静态类（静坐/站立/躺卧）信号平缓。时域方差可大致区分动静，但静坐与站立、上楼与下楼的波形高度相似，仅靠时域统计量难以精确区分。

### 图 2 — STFT 频带能量时间演进

![STFT频带能量](figures/02_STFT频带能量时间演进.png)

将 STFT 按四个频带（DC–2Hz、2–5Hz、5–10Hz、10–25Hz）聚合为能量值，在时间轴上画成**折线图**（替代传统 2D 热力图）。实线=行走，虚线=静态动作。

行走的 2–5 Hz（步频区间）能量持续偏高，静态动作几乎为零。折线图可直接读出精确数值，避免了热力图中 colorbar 各自归一化导致的跨图不可比问题。

### 图 3 — FFT 平均幅度谱叠加对比

![FFT幅度谱](figures/03_FFT平均幅度谱对比.png)

六种动作在 acc_x / y / z 三轴上的平均 FFT 幅度谱叠加。行走类在 1–3 Hz 有一致的步频主峰；静态类能量集中在 DC 附近（< 2 Hz）。频域能精确揭示"能量集中在哪些频率"，这是时域无法提供的信息。

### 图 4 — 频带能量堆叠柱状图

![频带能量](figures/04_频带能量分布.png)

将 0–25 Hz 分为 5 个频段，展示各动作在不同频段的能量占比。静态动作 >85% 能量在 DC–2 Hz；行走的 2–5 Hz 占比约 30%；上楼的 5–15 Hz 占比显著高于平地行走。

### 图 5 — 频谱特征空间散点图

![频谱散点](figures/05_频谱特征散点图.png)

以 acc_x 主峰值频率为横轴，谱熵 / 谱能量为纵轴。六类动作在频域空间中自然聚为可分离的簇——动态类高主频+高谱熵，静态类低主频+低谱熵。两个频域特征即可实现 >85% 的动/静二分类。

### 图 6 — 特征重要性排名

![特征重要性](figures/06_特征重要性排名.png)

决策树 Gini 重要性 Top-20。红色 = 频域特征，蓝色 = 时域特征。前 20 中频域特征占绝大多数，排名第 1 的 `acc_x_主峰频率` 重要性远超其他特征。

### 图 7 — 混淆矩阵

![混淆矩阵](figures/07_混淆矩阵.png)

测试集（2947 窗口）混淆矩阵。行走、静坐、站立、躺卧四类的召回率 > 85%。上楼与下楼之间有约 15% 的混淆（频域特征高度重叠）。

## 消融实验结果

| 特征集 | 维度 | 测试准确率 |
|--------|:----:|:----------:|
| 纯时域（均值、方差、峰峰值、过零率） | 24 | ~74% |
| 纯频域（主峰频率、谱熵、频带能量等） | 48 | ~76% |
| **融合（时域 + 频域）** | **72** | **~81%** |

频域特征相比纯时域提供 2–7 个百分点的提升。频域捕获了周期性结构（步频 ~1.5 Hz）和频谱复杂度（谱熵）——这些信息在时域统计量中不可见。

---

## 关键技术点

- **FFT 单边幅度谱**：`|X[k]| / N`，去直流后归一化，提取主峰频率、均值频率、中位数频率、谱熵、频带能量
- **STFT**：32 点汉明窗 → 频带聚合 → 折线图可视化（避免热力图跨图不可比）
- **决策树**：GridSearchCV 调参，Gini 不纯度分裂，特征重要性可解释
- **消融实验**：分别使用纯时域/纯频域/融合特征训练，量化频域贡献
