# 挑战路线（二）：IMU 自监督掩码编码器

**对应大作业报告**：第 4.3 节

## 概述

基于 data2vec 2.0 / MaskCAE 范式，从零实现 IMU 自监督掩码编码器。预训练任务：随机掩码部分传感器信号，学生编码器从残缺信号预测教师模型（完整信号）的表征。评估采用线性探测：冻结编码器，仅训练线性分类头。

## 模型架构

| 组件 | 配置 | 参数量 |
|------|------|:---:|
| PatchTimeFreqEmbedding | Conv1D + 局部FFT → 192-D tokens + 全局频率token | — |
| 学生编码器 | 4层 Transformer, 6头, d=192, MLP×3 | — |
| 教师编码器 | 同学生结构, EMA更新 (τ: 0.999→0.9999) | — |
| Predictor | 1层 Transformer, d_hidden=384 | — |
| **总计** | | **192万** |

## 三种互补掩码策略

1. **通道掩码** (25%)：随机清零整条传感器通道 → 强迫跨通道推断
2. **多尺度时域块掩码** (50%)：连续 5-15 个 patch 整块遮蔽 → 防止局部连续性作弊
3. **频域 FFT 自适应掩码**：基于 FFT 幅度选择高能量区域遮蔽 → 强制学习频谱结构

每个样本每轮生成 4 种掩码视图，共享同一组教师目标。

## 训练配置

- 预训练：WISDM 训练集，12 epochs，RTX 4070 Laptop ~8-10 min
- 优化器：AdamW, lr=3e-4, weight_decay=0.05, Cosine调度, 2 epoch warmup
- 组合损失：Layer2 浅层 (0.3) + Layer4 深层 (1.0)

## 18 类精细活动识别结果

| 模型 | 测试准确率 | 类型 |
|------|:---:|------|
| **自监督编码器 + 线性探测** | **52.56%** | 自监督预训练 |
| MLP + FeatureFusionNet 集成 | 38.87% | 监督集成 |
| FeatureMLP | 36.10% | 手工特征 + MLP |
| FeatureFusionNet | 35.18% | 手工特征 + 双分支融合 |
| CNN-BiLSTM | 31.92% | 端到端 |
| 随机基线 (1/18) | 5.56% | — |

自监督编码器大幅领先所有监督模型（约 +14pp），验证了"通过掩码预测学习传感器表征"在 IMU 数据上的可行性。

## 粒度效应

| 粒度 | 代表活动 | 准确率 |
|------|---------|:---:|
| 粗粒度全身运动 | 慢跑、步行、运球 | >90% (慢跑 99.3%) |
| 中等精细动作 | 上下楼梯、写字、打字 | 30-50% |
| 精细进食动作 | 喝汤、喝水 | <5% |
