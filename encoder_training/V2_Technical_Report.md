# WISDM 掩码预测编码器 V2 技术报告

## 1. 实验目标

在 WISDM 智能手表+手机活动和生物特征数据集上，训练一个自监督编码器用于人体活动识别（HAR）。核心挑战在于该数据集 18 种活动中，有 12 种是手部精细动作（打字、刷牙、进食等），传统方法对此区分度不足。

## 2. 方法

### 2.1 整体架构

基于 data2vec 2.0 / MaskCAE 范式，采用师生网络架构：

- **y-encoder（教师）**: 对完整原始信号编码，EMA 更新，提供预测目标
- **x-encoder（学生）**: 对掩码信号编码，插入可学习 [MASK] token，全注意力机制
- **Predictor**: 从 x-encoder 输出预测掩码位置的 y-encoder 表征
- **SigReg**: Sigmoid + 方差正则 + 熵正则，防止表征塌缩

### 2.2 数据选择

仅使用**手表数据**（加速度计 xyz + 陀螺仪 xyz，共 6 通道）。原因：18 种活动中 12 种为手部动作，手表直接捕捉手腕运动，手机在口袋中无用。

### 2.3 Patch 化与时频融合

- Patch size = 10 timesteps（0.5s @ 20Hz），每窗口 20 个 patch
- Conv1d stem 提取时域特征（embed_dim/2）
- 每 patch 局部 FFT 幅度 → Linear 投影（embed_dim/2）
- Concat + Linear 融合 → embed_dim 维 token
- 额外全局频率 token：全信号 FFT(101 bins) → Linear

### 2.4 掩码策略（Multi-Mask, M=4）

每个样本生成 4 种不同掩码视图，共享一个教师目标：

| 掩码类型 | 概率 | 说明 |
|---------|------|------|
| 通道掩码 | 25% | 随机抹除 1-2 个完整通道 |
| 多尺度时域块掩码 | 50% | 40% 掩码率，块大小 3-8(60%) / 10-18(30%) / 20-30(10%) |
| 频域 FFT 掩码 | 25% | 偏向中高频段的频率 bin 置零 |

### 2.5 损失函数

总损失 = L_pred + L_sigreg + 0.1 × L_lmm + 0.15 × L_supcon

- **L_pred**: MSE（预测表征 vs 教师表征），多层目标加权（Layer2:0.3, Layer4:1.0）
- **L_sigreg**: Sigmoid 方差 + 熵正则化
- **L_lmm**: Log-Magnitude Mean 损失——重建掩码 patch 信号，在 FFT 对数幅度域计算 MSE，强制编码器关注高频细节
- **L_supcon**: 监督对比损失——同标签样本在 CLS 投影空间中聚合（τ=0.07, 64 维投影）

## 5. 分析

### 5.1 有效的地方

- **Multi-mask (M=4)** 显著提升了数据利用率，每个样本提供 4 倍监督信号
- **x-encoder [MASK] token** 使可见/掩码区域之间有完整自注意力，表征质量明显提升
- **SupCon 损失** 有效拉近了同类别 CLS 表征（L_supcon 从 1.48 降至 0.16）
- **大动作识别** 已达到或接近可用水平

### 5.2 仍存在的问题

1. **饮食类动作极差**：喝汤/喝水准确率不到 4%，这些动作手腕运动幅度极小，6 通道加速度+陀螺仪数据可能本质上不足以区分
2. **L_lmm 上升**：频谱损失权重 0.1 太低，LightReconHead（128 维隐藏层）太弱，不足以对总损失产生实质性影响
3. **过拟合**：训练 97.7% vs 验证 63.3%，分类器在训练集上几乎完美但泛化差
4. **验证/测试域偏移**：验证 63.3% vs 测试 52.6%，2 个验证被试与 2 个测试被试的分布差异较大

### 5.3 改进方向

1. **增大 L_lmm 权重**至 0.3-0.5，或使用更深的 LightReconHead
2. **增加 phone 数据**作为额外通道，对进食类动作可能提供补充信息（手腕角度变化等）
3. **更强正则化**：分类器添加更大的 Dropout(0.5-0.7)、权重衰减
4. **更长的预训练**：12 epochs 对于 7802 个样本的学习可能不够充分
5. **数据增强**：时间拉伸、噪声注入等，提高饮食类样本的多样性
6. **对比学习负样本挖掘**：对易混淆类别对（soup/drink, chips/sandwich）施加更大的对比损失权重

## 6. 结论

V2 通过引入 multi-mask 训练、x-encoder mask token、LMM 频谱损失和监督对比学习，将测试准确率从 43.5% 提升至 52.6%（+9pp）。大动作识别效果显著，但精细进食动作的区分仍是核心瓶颈。频谱辅助损失和监督对比学习的潜力尚未完全释放，后续需要调整损失权重、加强正则化并考虑引入手机传感器数据。
